Ajax

Ajax是什么

全称： Asynchronous JavaScript And XML ,意为异步的JS和XML

作用： 利用AJAX可以在浏览器中向服务器发送异步请求，能够无刷新地获取数据

AJAX不是新的编程语言，而是一种将现有的标准组合在一起使用的新方式。

XML

  1. XML是可扩展标记语言，不像HTML中为预定义标签，XML中全都是自定义标签。
  2. XML被设计用来传输和存储数据
  3. 一开始Ajax返回的是XML格式数据，后来变为JSON，因为
    • JSON数据格式简洁
    • 数据转换容易且快速灵活

AJAX的特点

AJAX 的优点

1) 可以无需刷新页面而与服务器端进行通信。
2) 允许根据用户事件来更新部分页面内容。

AJAX 的缺点

  1. 没有浏览历史，不能回退
  2. 存在跨域问题
  3. SEO 不友好

HTTP

HTTP（hypertext transport protocol）协议【超文本传输协议】，协议详细规定了浏览器和万维网服务器之间相互通信的规则。

格式： 请求行+请求头+空行+请求体

1.请求行

请求类型+URL+HTTP协议的一个版本

POST/GET+/…..+HTTP/1.1

2.请求头

HOST + : + 空格 + 值

Cookie:

Content-type:

User-Agent:

3.空行

4.请求体

GET请求，请求体为空；

格式： 响应行+响应头+空行+响应体

1.响应行
